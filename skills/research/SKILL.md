# Research

State the question and the decision it informs. Prefer source code, official
documentation, standards, and verified runtime evidence. Separate facts from
inferences, cite each material claim, and report uncertainty.

Keep the result conversational unless the assigned spec requires durable
evidence; then add a concise, linked finding in that spec. Research does not
make product decisions or create a parallel knowledge base.
