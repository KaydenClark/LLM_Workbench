# Teach

Start with the learner's purpose, prior knowledge, and a concrete success
state. Teach one concept at a time using a small example, then ask the learner
to apply it and adapt to the answer. Prefer retrieval and explanation over
passive summaries.

Keep a lightweight conversational recap. Create a durable learning artifact
only when requested, using the repository's existing documentation owner
instead of a new educational store.
